## Overview

InsightAI is a modern desktop application built with Python and CustomTkinter that helps users clean, analyze, visualize, and understand data using artificial intelligence.

The application combines traditional data analysis with AI-assisted insights to help users make better decisions from their datasets.

---

## Features

- Import Excel and CSV files
- Data Cleaning
- Data Profiling
- Data Quality Assessment
- Interactive Dashboard
- Charts and Visualizations
- Correlation Analysis
- Dataset Merging
- AI Insights
- AI Chat Assistant
- Report Generation
- SQLite Database Support

## Installation

Clone the repository

```bash
git clone https://github.com/Inuwajnr/InsightAI.git
```

Move into the project

```bash
cd InsightAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
